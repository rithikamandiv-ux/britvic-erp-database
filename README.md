# BritVic Enterprise Resource Management System

## 📌 Description
This project is a relational database system designed to manage enterprise resources within an organization.  
It supports core business operations such as asset management, employee management, inventory tracking, production planning, and purchasing processes.

The system was developed as part of coursework to demonstrate database design, normalization, and SQL implementation.

---

## ⚙️ Technologies Used
- MySQL
- phpMyAdmin
- SQL (DDL & DML)

---

## 🗄️ Database Structure

The database includes the following main modules and tables:

### Core Modules:
- Employee & Department Management
- Asset & Asset Maintenance
- Inventory & Materials
- Production Planning
- Purchasing & Procurement
- Sales & Orders

### Key Tables:
- Employee
- Department
- Asset
- AssetMaintenance
- Inventory
- Material
- ProductionPlan
- PurchaseOrder
- SalesOrder

---

## 🚀 Key Features

- Manage employee and departmental data
- Track company assets and maintenance records
- Handle inventory and material requests
- Support production planning workflows
- Manage purchase orders and supplier interactions
- Record sales orders and business transactions
